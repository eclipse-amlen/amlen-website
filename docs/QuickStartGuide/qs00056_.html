Installing the Eclipse Amlen RPM on a host OS

Install the Eclipse Amlen RPM directly on your host operating system (OS).

Before you install the Eclipse Amlen on your host OS, you must provision your OS and prepare the Eclipse Amlen .tz files. For more information, see Provisioning your operating system and Preparing the Eclipse Amlen .tz packages for installation.

Once you have installed Eclipse Amlen, you must set and accept the license. Once you have accepted the license, you can start using Eclipse Amlen. For more information about setting and accepting the license, see Configuring the licensed usage for Eclipse Amlen and accepting the license by using REST Administration APIs.

.

To run Eclipse Amlen services, you need to install the Eclipse Amlen RPM on a supported host environment. After successful deployment, you can start to use Eclipse Amlen to handle connected clients and devices and also to process messages. You can use the Amlen WebUI to administer the Eclipse Amlen messaging server.

  • Installing the Eclipse Amlen messaging server RPM
    Install the Eclipse Amlen messaging server RPM. The Eclipse Amlen messaging server handles connected clients and devices, and processes messages.
  • Installing the Amlen WebUI RPM
    Install the Amlen WebUI RPM. You can then use the Amlen WebUI to administer the Eclipse Amlen messaging server.
