Installing Eclipse Amlen

You can install Eclipse Amlen in Docker images or as RPMs directly on your host operating system (OS).

The installation process of Eclipse Amlen involves the following processes:
  1. Provision your OS.
  2. If you are using Docker, install Docker on your OS.
  3. Prepare the Eclipse Amlen tz packages for installation.
  4. Install Eclipse Amlen and accept the license agreement.
  • Provisioning your operating system
    Before you install Eclipse Amlen, you must provision an operating system in one of the supported operating environments.
  • Installing Docker on your operating system
    After you have provisioned your supported operating system (OS), you can install Docker on the OS.
  • Preparing the Eclipse Amlen .tz packages for installation
    The .tz files contain the files that are required to install Eclipse Amlen.
  • Installing Eclipse Amlen
    You can install Eclipse Amlen in Docker containers or as RPMs directly on your host operating system. On Red Hat systems, you must have an active Red Hat subscription to install Eclipse Amlen in Docker containers or as RPMs directly on your host operating system. You do not need such a subscription on CentOS.
  • Logging in to the Amlen WebUI
    Log in to the Amlen WebUI to verify the installation of Eclipse Amlen and to administer the Eclipse Amlen messaging server.
  • Verifying the installation of Eclipse Amlen
    You can verify the installation of Eclipse Amlen by using the Messaging Tester sample application.
