Installing Docker on your operating system

After you have provisioned your supported operating system (OS), you can install Docker on the OS.

Before you install and start the Docker service on your OS, ensure that the OS has been provisioned on an appropriate supported environment. For more information, see Provisioning your operating system.

To install Docker on your OS, see the Docker installation instructions on the website for your OS:
  • CentOS
  • Red Hat

After the Docker service is installed, you can prepare the .tz files. For more information, see Preparing the Eclipse Amlen .tz packages for installation.
