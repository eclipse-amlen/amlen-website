Installing Eclipse Amlen

You can install Eclipse Amlen in Docker containers or as RPMs directly on your host operating system. On Red Hat systems, you must have an active Red Hat subscription to install Eclipse Amlen in Docker containers or as RPMs directly on your host operating system. You do not need such a subscription on CentOS.

After you have installed the Eclipse Amlen server, you must set the license type and accept the license agreement by using a REST Administration API. Until the license is accepted, the Eclipse Amlen server remains in maintenance mode. For more information about using a REST Administration API to set the licensed usage and accept the license agreement, see Configuring the licensed usage for Eclipse Amlen and accepting the license by using REST Administration APIs.

    Install Eclipse Amlen in Docker containers by building and deploying the Docker images that you need to run Eclipse Amlen.
  • Installing the Eclipse Amlen RPM on a host OS
    Install the Eclipse Amlen RPM directly on your host operating system (OS).
