Preparing the Eclipse Amlen .tz packages for installation

The .tz files contain the files that are required to install Eclipse Amlen.

Before you prepare the .tz file, ensure that the OS has been provisioned on an appropriate supported environment. For more information, see Provisioning your operating system.

  1. Copy the following files to the OS on which you are installing Eclipse Amlen:
    • EclipseAmlenServer-<build_info>.tz
    • EclipseAmlenWebUI-<build_info>.tz
  2. Create subdirectories for the .tz files.
    For example, create a subdirectory called imaserver for the EclipseAmlenServer-<build_info>.tz file and a subdirectory called imawebui for the EclipseAmlenWebUI-<build_info>.tz file
  3. Unpack the contents of the EclipseAmlenServer-<build_info>.tz file into the imaserver subdirectory.
  4. Unpack the contents of the EclipseAmlenWebUI-<build_info>.tz file into the imawebui subdirectory.
If you are using Docker, you can proceed to install Eclipse Amlen in Docker containers. For more information, see Installing Eclipse Amlen in Docker containers. Alternatively, you can install the RPM directly on your host OS. For more information, see Installing the Eclipse Amlen RPM on a host OS.
