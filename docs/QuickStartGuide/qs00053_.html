Logging in to the Amlen WebUI

Log in to the Amlen WebUI to verify the installation of Eclipse Amlen and to administer the Eclipse Amlen messaging server.

Ensure that the Amlen WebUI and Eclipse Amlen messaging server are running.

You can use the URL http://<server:Port> to connect to the Amlen WebUI. This task uses https://10.10.2.149:9087/ to connect to the Amlen WebUI. To use this URL, your Amlen WebUI container must be running on the host with IP address 10.10.2.149.

You need to accept an Amlen WebUI certificate and an Eclipse Amlen server certificate before you can start using the Amlen WebUI. Both of these certificates must be changed if you move from a non-production to a production environment.

  1. Open an Internet Explorer 11, Firefox, or Chrome browser and connect to the Amlen WebUI by entering the following URL:
    10.10.2.149:9087
  2. Accept the Amlen WebUI certificate so that you can view the login page.
  3. Log in and accept the Eclipse Amlen server certificate in one of the following ways:
    • If you are using Internet Explorer 11, complete the following steps to log in and accept the Eclipse Amlen server certificate:
      1. At the login prompt, type the default ID and password (admin/admin ) and click Log in.
      2. After you are logged in, follow the browser prompts to accept the certificate.
      3. The Common configuration and customization tasks section of the IBM Watson IoT Platform - Message Gateway Web UI home page is displayed.
    • If you are using Firefox or Google Chrome, complete the following steps to log in and accept the Eclipse Amlen server certificate:
      1. At the login prompt, type the default ID and password (admin/admin ) and click Log in. The error Cannot connect to the IBM Message Gateway server is displayed.
      2. Click Click here in the message text. An error page is displayed that indicates that you need to accept the certificate.
      3. Follow the browser prompts to accept the certificate. The About IBM Watson IoT Platform - Message Gateway page is displayed in a new browser window.
      4. Return to the IBM Watson IoT Platform - Message Gateway Web UI page. If the error box is still displayed, refresh the page.
      5. The Common configuration and customization tasks section of the IBM Watson IoT Platform - Message Gateway Web UI home page is displayed.
    Note: You can replace the default certificate with your own appropriately signed certificate and then you do not have to keep accepting certificates. For information about specifying your own certificate to replace the default certificate, see Changing the Amlen WebUI security certificate.
You can use the Amlen WebUI to verify your installation of Eclipse Amlen and to administer the Eclipse Amlen messaging server. For more information about verifying your installation, see Verifying the installation of Eclipse Amlen. For more information about using the Amlen WebUI to administer the Eclipse Amlen messaging server, see Administering.
Note: The Amlen WebUI can administer the messaging server only after the license has been accepted on the messaging server. Use the Eclipse Amlen REST API POST method to accept the license on the messaging server. For more information about accepting the license by using the Eclipse Amlen REST API POST method, see Configuring the licensed usage for Eclipse Amlen and accepting the license by using REST Administration APIs.
