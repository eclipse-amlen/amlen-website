Provisioning your operating system

Before you install Eclipse Amlen, you must provision an operating system in one of the supported operating environments.

In order to install Eclipse Amlen, you must first provision a supported operating system. Supported operating systems are CentOS/RHEL/Alma 7 or 8 on x86_64 and recent Fedora in x86_64 (which version of Fedora builds are provided for will change over time due to the short life of Fedora versions). You can provision the operating system in any of the supported environments:
  • Cloud provider: IBM Cloud, Microsoft Azure, or Amazon AWS.
  • Bare metal system in a supported cloud provider or in your own location.
  • VMware ESXi in a supported cloud provider or in your own location.
For information about how to provision the operating system on a supported cloud provider, see the setup instructions on the cloud provider's website:
  • IBM Cloud
  • Microsoft Azure
  • Amazon AWS
  • VMware ESXi
