Configuring Web UI users by using the Amlen WebUI

System administrators can configure Web UI users to administer the Eclipse Amlen only by using the Web UI User Management page in the Amlen WebUI.

You can configure Web UI users by using the Amlen WebUI. You cannot configure Web UI users by using REST Administration APIs.

For more information about the properties that you must specify when you configure a Web UI user, see Configuring Eclipse Amlen Web UI users.

Note: The user admin is a special user. This user cannot be deleted or have its role changed.
  1. Go to the Web UI User Management page in one of the following ways:
    • From the Home tab, go to the Web UI Users link in the Create users pane.
    • From the Web UI menu, select Web UI Users.
  2. Choose an action:
    • To add a Web UI user, click the Add icon.
    • To delete a user, select the Web UI user and click the Delete icon.
    • To edit a user, select the Web UI user and click the Edit icon.
    • To reset the password of a user, select the Web UI user and click Other Actions > Reset Password.
