Changing the Amlen WebUI security certificate

You can change the security certificate that is used by the Amlen WebUI by using the Web UI Settings page.

The Amlen WebUI contains a default security certificate which is used by the Amlen WebUI. You can upload a new security certificate to replace the default certificate, or a certificate that you previously uploaded. Replacing the default certificate gives a better level of security.

Important: The Web UI certificate must be either PEM (with either .pem, .crt, or .key file extensions) or PKCS12 (with a .p12 file extension) format.
  1. Go to the Web UI Settings page:
    • From the Web UI menu, click Web UI Settings. The Web UI Settings page is displayed.
  2. From the Security Certificate pane, click Edit Certificate
  3. Enter the certificate details, and click Save
