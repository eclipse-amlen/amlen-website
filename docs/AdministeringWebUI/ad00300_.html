Configuring Eclipse Amlen Web UI users

You can configure Web UI users to administer Eclipse Amlen only by using the Amlen WebUI.

Web UI users can administer Eclipse Amlen by using the Amlen WebUI.

When you create Web UI users, you must specify the following properties:
  • User ID
    The name that is used by the user to log in.
    The name must be unique. The name can only contain the letters A - Z and a - z, the numbers 0 - 9, and the following characters:
    - _ . +
  • Role
    The role defines the type of Web UI user:
    • System administrators
    • Messaging administrators
    • Web UI users
    For more information about types of Web UI users, see Eclipse Amlen Web UI users and roles.
  • Password
    The password for the user.
    The password cannot contain leading or trailing spaces.
    The password cannot contain the following character:
    "

For more information about configuring Web UI users by using the Amlen WebUI, see Configuring Web UI users by using the Amlen WebUI.
