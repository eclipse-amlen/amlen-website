Eclipse Amlen Web UI users and roles

There are three types of Web UI user roles: system administrators, messaging administrators, and Web UI users.

The system administrator role has access to all of the view and configuration tasks provided in the Web UI. The system administrator can access Eclipse Amlen using the web-based user interface. The following table shows the actions which system administrators, messaging administrators, and Web UI users are authorized to perform.
