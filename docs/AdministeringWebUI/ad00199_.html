The Amlen WebUI

The Amlen WebUI can be used by system administrators and messaging administrators to administer Eclipse Amlen.

The Amlen WebUI can be accessed by using any of the supported browsers. The browser must have JavaScriptâ„¢ enabled.

You can connect to the Amlen WebUI by entering the following URL:

Note: This URL uses the default port value. If you changed the port value, replace the 9087 section of the URL with your port number.

https://IP_Address:9087

Where:
  • IP_Address
    Specifies the IP address of the host where the Web UI application is running.

The Amlen WebUI status menu

From the status menu, you can view the status of the Eclipse Amlen server. You can also view the status of your high availability and cluster configuration.

The state of the Eclipse Amlen server changes which actions are available in the Amlen WebUI:
  • When the server is in maintenance mode, you cannot use any monitoring actions.
  • When the server is stopped, you can use only a limited number of configuration actions.
  • When the server is restarting, you cannot use any actions until the server is started.
  • When the server is in a highly available configuration, and is the secondary node, you can use only a limited number of configuration actions. You must make configuration changes with the primary node.
For more information about the different server states, and how to change between the states, see Starting, stopping, restarting and viewing the status of the Eclipse Amlen server, and Maintenance mode.

The Amlen WebUI user menu

The icon for the user menu is your user name. From the user menu, you can complete the following tasks:
  • Change your password
  • Logout

The Amlen WebUI help menu

The icon for the help menu is a question mark in a circle. From the help menu, you can complete the following tasks:
  • Access the product help by clicking Help
  • Restore the common configuration and customization tasks to the home page by clicking Restore Tasks on Home Page
  • View the available software license agreements by clicking About and then selecting the link for the license that you would like to view.
