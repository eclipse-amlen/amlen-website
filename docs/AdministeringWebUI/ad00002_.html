Administering the Web UI

System administrators can administer the Web UI by using the Amlen WebUI.

  • Eclipse Amlen Web UI users and roles
    There are three types of Web UI user roles: system administrators, messaging administrators, and Web UI users.
  • Configuring Eclipse Amlen Web UI users
    You can configure Web UI users to administer Eclipse Amlen only by using the Amlen WebUI.
  • The Amlen WebUI
    The Amlen WebUI can be used by system administrators and messaging administrators to administer Eclipse Amlen.
