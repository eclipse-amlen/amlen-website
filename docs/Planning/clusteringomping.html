Clustering: Testing multicast connectivity between servers in a cluster

You can use the omping tool to test basic multicast connectivity between Eclipse Amlen servers in a cluster.

The omping tool is available in the omping package that is provided with the base operating system. The tool allows you to test basic multicast connectivity between the Eclipse Amlen servers in a cluster. For more information about the omping tool, read the man page.

See Using omping to test multicast connectivity in clusters for information about using the omping tool on Eclipse Amlen.
