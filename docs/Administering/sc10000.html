Scenarios to connect WebSphere Application Server and Eclipse Amlen

These scenarios use sample applications to demonstrate the use of WebSphere® Application Server connected to Eclipse Amlen. You can set up a copy of the sample installation as described in each of the scenarios. You can use the sample applications that are provided with each scenario to verify your progress through each stage. The scenarios are available in the IBM® Product Connectivity Scenarios and Patterns information center.

The first scenario shows you how to connect Java™ EE applications on WebSphere Application Server to Eclipse Amlen, in order to consume and work with messages from Eclipse Amlen. Starting with an existing Eclipse Amlen installation, this scenario leads you through the key tasks that are required to install and connect WebSphere Application Server. For more information, see Scenario: Connecting WebSphere Application Server to Eclipse Amlen.

The second scenario shows you how to connect mobile devices to Java EE applications on WebSphere Application Server by using the MQTT protocol. Starting with an existing Eclipse Amlen and WebSphere Application Server installation, this scenario leads you through the key tasks that are required to connect a mobile device to WebSphere Application Server. For more information, see Scenario: Adding a mobile application with Eclipse Amlen toWebSphere Application Server.
