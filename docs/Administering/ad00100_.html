Working with external resources

You can configure Eclipse Amlen to work with external resources by using MQ Connectivity or the Eclipse Amlen JMS resource adapter.

The MQ Connectivity feature can be used to connect to existing IBM® MQ networks. The JMS resource adapter feature can be used to connect to a WebSphere® Application Server or a WebSphere Application Server cluster.

For more information about the uses of MQ Connectivity and the JMS resource adapter, see Architecture Overview.

  • MQ Connectivity
    Eclipse Amlen can receive messages that are published to Eclipse Amlen topics, or sent to Eclipse Amlen queues and can forward the messages to IBM MQ queues or topics. Eclipse Amlen can also receive messages that are published to IBM MQ topics, or sent to IBM MQ queues, and can forward the messages to Eclipse Amlen topics or queues.
  • Eclipse Amlen resource adapter
    The Eclipse Amlen resource adapter allows applications that run on WebSphere Application Server to connect to Eclipse Amlen directly. The Eclipse Amlen resource adapter supports inbound and outbound communication.
  • Using VMware vSphere to handle Eclipse Amlen failover
    You can use VMware vSphere to resolve Eclipse Amlen failover.
