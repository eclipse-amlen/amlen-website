Setting and viewing the host IP address or host name of the JVM remote debugger of the protocol plug-in server

System administrators can set and view the host IP address or host name of the JVM remote debugger of the protocol plug-in server by using REST Administration APIs.

  • Setting the host IP address or host name of the JVM remote debugger of the plug-in server
    System administrators can set the host IP address or host name of the JVM remote debugger of the plug-in server by using the Eclipse Amlen REST API POST method.
  • Viewing the host IP address or host name of the JVM remote debugger of the plug-in server
    System administrators can view the host IP address or host name of the JVM remote debugger of the plug-in server by using the Eclipse Amlen REST API GET method.
