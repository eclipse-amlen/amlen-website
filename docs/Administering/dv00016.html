Viewing the JVM maximum heap size of the plug-in server

System administrators can view the JVM maximum heap size of the plug-in server by using the Eclipse Amlen REST API GET method.

To view the JVM maximum heap size of the plug-in server, use the Eclipse Amlen REST API GET method with the following Eclipse Amlen URI:

http://<admin-endpoint-IP:Port>/ima/v1/configuration/PluginMaxHeapSize

The following example shows a GET method to view the JVM maximum heap size of the plug-in server by using cURL: 

curl -X GET http://127.0.0.1:9089/ima/v1/configuration/PluginMaxHeapSize

The following example shows a successful example response to the GET method that was used to view the JVM maximum heap sizeof the plug-in server:


     {   
       "Version": "v1",
       "PluginMaxHeapSize": 1024
      }
