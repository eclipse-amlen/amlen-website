Configuring cluster membership by using the Amlen WebUI

System administrators can configure cluster membership of Eclipse Amlen servers by using the Cluster Membership page in the Amlen WebUI.

You can configure the cluster membership of an Eclipse Amlen server by using the Amlen WebUI, or by using REST Administration APIs. For more information about using REST Administration APIs to configure cluster membership, see Configuring cluster membership by using REST Administration APIs.

For more information about the components that you must specify when you configure cluster membership, see Configuring the cluster membership of an Eclipse Amlen server.

  1. Go to the Cluster Membership page:
    • From the Cluster menu, click Join/Leave.
  2. Click Edit in the Configuration pane and enter configuration details.
  3. Click Save.
  4. Choose an action:
    • To add the server to the cluster, click Restart server and join cluster in the Status pane.
    • To reset cluster configuration settings to default values, select Reset in the Configuration pane.
  5. To remove the server from the cluster, click Leave cluster in the Status pane.
