Configuring licensed usage and accepting the license for Eclipse Amlen

You can configure the licensed usage and accept the license for Eclipse Amlen by using a single REST Administration API.

The Eclipse Amlen server will start in maintenance mode until the license is accepted. You can view the current licensed usage by using the Eclipse Amlen Web UI or by using a REST Administration API.

The licensed usage types are:
Developers
For use in a development and test environment.
Non-Production
For use in a development and test environment.
Production
For use in a production environment.
IdleStandby
For use in a high availability environment.

If you change the licensed usage of Eclipse Amlen, you must accept the license.

For more information about configuring the licensed usage and accepting the license for Eclipse Amlen by using a single REST Administration API, see Configuring the licensed usage for Eclipse Amlen and accepting the license by using REST Administration APIs.

For more information about viewing the license for Eclipse Amlen by using the Eclipse Amlen Web UI, see Viewing the licensed usage for Eclipse Amlen by using the Amlen WebUI
