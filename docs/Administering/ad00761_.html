Configuring OAuth profiles by using the Amlen WebUI

System administrators can configure OAuth profiles by using the Security Settings page in the Amlen WebUI.

You can configure OAuth profiles by using the Amlen WebUI, or by using REST Administration APIs. For more information about using REST Administration APIs to configure OAuth profiles, see Creating and updating an OAuth profile by using REST Administration APIs.

For more information about the components that you must specify when you configure LTPA profiles, see Configuring OAuth.

  1. Go to the Security Settings page in one of the following ways:
    • From the Home tab, click Server Security in the Secure your environment pane.
    • From the Server menu, click Security Settings.
  2. Choose an action from the OAuth Profiles pane:
    • To add an OAuth profile, click the Add icon.
    • To delete an OAuth profile, select the profile and click the Delete icon.
      Note: You cannot delete an OAuth profile that is associated with a security profile. First, remove the OAuth profile from the associated security profile, and then delete the OAuth profile.
    • To edit an OAuth profile, select the profile and click the Edit icon.
Associate the OAuth profile with a security profile. For more information, see Configuring security profiles.
