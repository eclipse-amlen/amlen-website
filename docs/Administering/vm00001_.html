Using VMware vSphere to handle Eclipse Amlen failover

You can use VMware vSphere to resolve Eclipse Amlen failover.

For more information about VMware documentation for ESXi 5.5, see chapter 2 "Creating and Using vSphere HAClusters" of the vSphere Availability documentation https://pubs.vmware.com/vsphere-55/topic/com.vmware.ICbase/PDF/vsphere-esxi-vcenter-server-55-availability-guide.pdf or you can find the most up-to-date technical documentation on the VMware Web site http://www.vmware.com/support.html

  1. On vSphere, create a HA cluster - for more information, see the vSphere Availability documentation.
  2. Configure SAN as a data store.
  3. Configure the Eclipse Amlen OVA so that it is part of the HA cluster.
