Configuring the server name

You can configure the server name either by using the Amlen WebUI or by using REST Administration APIs.

Configuring the server name helps you to identify the Eclipse Amlen server on which you are operating.

For more information about configuring the server name by using the Amlen WebUI, see Configuring the server name by using the Amlen WebUI.

For more information about configuring the server name by using REST Administration APIs, see Configuring the server name by using REST Administration APIs.
