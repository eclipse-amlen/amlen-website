Disabling cluster membership of Eclipse Amlen servers in a high availability pair by using REST Administration APIs

You can disable cluster membership of Eclipse Amlen servers in a high availability (HA) pair by using REST Administration APIs..

Ensure that the primary and standby servers in the HA pair are active.

System administrators can disable cluster membership of Eclipse Amlen servers in an HA pair by using REST Administration APIs, or by using the Amlen WebUI. For more information about using the Amlen WebUI to disable cluster membership of servers in an HA pair, see Disabling cluster membership of Eclipse Amlen servers in a high availability pair.

  1. Disable cluster membership on the primary server only by setting "EnableClusterMembership": false

    For more information about using the Eclipse Amlen REST API POST method to configure cluster membership, see Configuring cluster membership by using REST Administration APIs.

  2. Both servers in the HA pair are automatically restarted.
