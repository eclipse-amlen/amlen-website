Scenarios

Scenarios are defined to highlight parameters that you might want to consider setting when you plan how to configure your Eclipse Amlen resource adapter.

You can use the scenarios to help you to identify the Eclipse Amlen resource adapter parameters that you might want to configure and tune to get up and running quickly.

  • Scenarios for configuring the Eclipse Amlen resource adapter for inbound communication
    There are some basic, underlying tasks that are common to each of the following scenarios that are associated with configuring the Eclipse Amlen resource adapter for inbound communication. The details for each task vary according to requirements. However, the following steps must be considered, understood, and completed to configure the resource adapter for inbound communication.
  • Security and the Eclipse Amlen resource adapter
    When you configure the Eclipse Amlen resource adapter, you must consider the configuration of the Eclipse Amlen server and the associated security requirements when you set the following activation specification and connection factory properties.
  • Scenarios to connect WebSphere Application Server and Eclipse Amlen
    These scenarios use sample applications to demonstrate the use of WebSphere® Application Server connected to Eclipse Amlen. You can set up a copy of the sample installation as described in each of the scenarios. You can use the sample applications that are provided with each scenario to verify your progress through each stage. The scenarios are available in the IBM® Product Connectivity Scenarios and Patterns information center.
