Configuring SNMP by using the Amlen WebUI

System administrators and messaging administrators can configure SNMP by using the SNMP Service panel in the Amlen WebUI.

You can enable or disable SNMP by using the Amlen WebUI, or by using REST Administration APIs. For more information about using REST Administration APIs to configure SNMP, see Configuring SNMP by using REST Administration APIs.

For more information about the components that you must specify when you configure a SNMP, see Configuring SNMP.

  1. Go to the Server>Server Control menu.
  2. In the SNMP Service section, review the SNMP enabled state.
    • To enable SNMP, select the Enable SNMP check box.
    • To disable SNMP, click the Stop the service link.
    • To start SNMP, click the Start the service link.
