Selecting an Eclipse Amlen server to manage by using the Amlen WebUI

When you log on to the Amlen WebUI, the Eclipse Amlen server name that is displayed next to the Server menu is the name of the server that was last accessed. You can select a different Eclipse Amlen server by using the Server menu.

To select a different server to manage, you must have two or more Eclipse Amlen servers configured in the Amlen WebUI. For information about adding an Eclipse Amlen server in the Web UI, see Managing a list of Eclipse Amlen servers by using the Amlen WebUI.

  1. Log on to the Amlen WebUI.
  2. Select the Server menu at the top of the Amlen WebUI page.
  3. From the Server panel, choose the Eclipse Amlen server that you want to manage by either:
    • Selecting the Eclipse Amlen server from the Server drop-down list on the Server panel.
    • Selecting the Eclipse Amlen server from the Recent Servers list on the Server panel.
      Note: The Recent Servers list shows a subset of Eclipse Amlen servers. Only the most recently accessed servers are displayed in this list.
