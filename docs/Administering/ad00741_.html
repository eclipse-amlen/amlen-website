Configuring security profiles by using the Amlen WebUI

System administrators can configure security profiles by using the Security Settings page in the Amlen WebUI.

If you need to enable Transport Layer Security (TLS) on an endpoint, you must create a certificate profile before you can create a security profile. The certificate profile defines the server certificate to use in the security profile. For more information about creating a certificate profile, see Configuring certificate profiles.

You can configure security profiles by using the Amlen WebUI, or by using REST Administration APIs. For more information about using REST Administration APIs to configure security profiles, see Creating and updating a security profile by using REST Administration APIs.

For more information about the components that you must specify when you configure security profiles, see Configuring security profiles.

  1. Go to the Security Settings page in one of the following ways:
    • From the Home tab, click Server Security in the Secure your environment pane.
    • From the Server menu, click Security Settings.
  2. Choose an action from the Security Profiles pane:
    • To add a security profile, click the Add icon.
    • To delete a security profile, select the profile and click the Delete icon.
    • To edit a security profile, select the profile and click the Edit icon.

Associate the security profile with an endpoint. For more information, see Configuring message hubs.
