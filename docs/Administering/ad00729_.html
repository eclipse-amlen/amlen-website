Configuring trusted certificates by using the Amlen WebUI

System administrators can configure trusted certificates by using the Security Settings page in the Amlen WebUI.

You can configure trusted certificates by using the Amlen WebUI, or by using REST Administration APIs. For more information about using REST Administration APIs to configure trusted certificates, see Creating and updating a trusted certificate by using REST Administration APIs.

  1. Go to the Security Settings page in one of the following ways:
    • From the Home tab, click Server Security in the Secure your environment pane.
    • From the Server menu, click Security Settings.
  2. From the Security Profiles pane, select a security profile.
  3. Click Other Actions.
  4. Select Trusted Certificates.
  5. Choose an action:
    • To upload a certificate to the truststore:
      1. Click Browse and navigate to the certificate file.
      2. Click Upload.
      Note: You can upload a maximum of 100 trusted client certificates for every security profile.
    • To delete a certificate:
      1. Select the certificate file name in the Certificates grid.
      2. Click the Delete icon.
  6. Click Close.
