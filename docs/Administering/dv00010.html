Configuring protocol plug-ins

The protocol plug-in provides the ability to extend the protocols that are supported by the Eclipse Amlen server. You can configure protocol plug-ins by using REST Administration APIs.

You can use protocol plug-ins to add industry-specific messaging protocols to Eclipse Amlen in a controlled way that does not compromise the security of the server environment. The protocol plug-in can also support legacy sensors that are not able to change to MQTT.
