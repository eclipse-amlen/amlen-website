Configuring an LDAP server by using the Amlen WebUI

System administrators and messaging administrators can configure an LDAP server connection by using the User Authentication page in the Amlen WebUI.

You can configure an LDAP server connection by using the Amlen WebUI, or by using REST Administration APIs. For more information about using REST Administration APIs to configure an LDAP server connection, see Configuring an LDAP server by using REST Administration APIs.

For more information about the components that you must specify when you configure an LDAP server connection, see Configuring LDAP for user authentication.
  1. Go to the User Authentication page in one of the following ways:
    • From the Home tab, click LDAP Configuration in the Configure IBM Watson IoT Platform - Message Gateway to accept connections pane.
    • From the Server menu, click User Authentication.
  2. Choose an action:

    From the External LDAP pane:

    • To edit an external LDAP server connection, click the Edit icon.
    • To enable the use of LDAP servers to manage messaging users and groups, click Enable LDAP.

      This option is only available if LDAP is disabled. If you enable LDAP, you cannot edit, or delete the Eclipse Amlen messaging users and groups.

    • To disable the use of an LDAP server to manage messaging and groups, click Disable LDAP.

      This option is only available if LDAP is enabled.

    • To test an LDAP server connection, select the LDAP server connection, and select Test connection from the Other Actions list.
    • To reset an LDAP server connection, select the LDAP server connection, and select Reset from the Other Actions list.
    Note: If you update the LDAP certificate, the new LDAP settings will be used the next time a client or connection is authenticated or authorized.
