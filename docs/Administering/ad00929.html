Managing multiple Eclipse Amlen servers from a single Amlen WebUI

You can use a single Amlen WebUI instance to manage and monitor multiple Eclipse Amlen servers that are version 2.0 or higher. The first time that you log into the Amlen WebUI, you are prompted for information about the first Eclipse Amlen server that you would like to manage. After your first login, you can add, edit or remove Eclipse Amlen servers to your managed server list by using the Server menu in the Amlen WebUI.

  • Configuring your first Eclipse Amlen server to manage by using the Amlen WebUI
    The first time that you log into the Amlen WebUI, you are prompted to submit information about the first Eclipse Amlen server that you would like to manage.
  • Managing a list of Eclipse Amlen servers by using the Amlen WebUI
    You can use the Server menu in the Amlen WebUI to update the list of Eclipse Amlen servers that you want to manage.
  • Selecting an Eclipse Amlen server to manage by using the Amlen WebUI
    When you log on to the Amlen WebUI, the Eclipse Amlen server name that is displayed next to the Server menu is the name of the server that was last accessed. You can select a different Eclipse Amlen server by using the Server menu.
