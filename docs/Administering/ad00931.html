Managing a list of Eclipse Amlen servers by using the Amlen WebUI

You can use the Server menu in the Amlen WebUI to update the list of Eclipse Amlen servers that you want to manage.

You can add new Eclipse Amlen servers to manage, update the configuration of existing Eclipse Amlen servers or remove Eclipse Amlen servers that you no longer want to manage by using the Server menu.

  1. Log on to the Amlen WebUI.
  2. Select the Server menu.
  3. From the Server panel, click Add or remove servers.
  4. From the Add or remove servers panel, choose an action:
    • To add an Eclipse Amlen server, click the Add icon. For more information about the fields that you can configure, see Configuring your first Eclipse Amlen server to manage by using the Amlen WebUI.
    • To remove an Eclipse Amlen server, select the server that you want to remove and click the Delete icon.
    • To update existing connection configuration details for an Eclipse Amlen server, select the server that you want to update and click the Edit icon.
