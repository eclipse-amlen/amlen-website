Configuring Federal Information Processing Standards (FIPS)

When you configure Eclipse Amlen security, you can enable or disable FIPS-compliant security by using either the Amlen WebUI or by using REST Administration APIs.

FIPS is a publicly announced standardization that is developed by the United States federal government for use in computer systems. For more information, see Federal Information Processing Standards Publications (FIPS PUBS).

When you change the FIPS settings for Eclipse Amlen, you must restart the Eclipse Amlen server.

For more information about configuring FIPS-compliant security by using the Amlen WebUI, see Configuring FIPS compliant security by using the Amlen WebUI.

For more information about configuring FIPS-compliant security by using REST Administration APIs, see Configuring FIPS-compliant security by using REST Administration APIs.
