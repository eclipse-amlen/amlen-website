Configuring cluster membership of Eclipse Amlen servers in a high availability pair by using REST Administration APIs

System administrators can configure cluster membership of Eclipse Amlen servers that are operating as a high availability (HA) pair.

You can enable and disable cluster membership on Eclipse Amlen servers in an HA pair.

When an HA pair is operating in a cluster, only the primary server participates in the cluster.
