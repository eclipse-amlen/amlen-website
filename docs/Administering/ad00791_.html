Viewing the licensed usage for Eclipse Amlen by using the Amlen WebUI

System administrators can view the licensed usage for Eclipse Amlen by using the Server Control page in the Amlen WebUI.

You can view the licensed usage for Eclipse Amlen by using the Amlen WebUI, or by using REST Administration APIs. For more information about using REST Administration APIs to view the licensed usage for Eclipse Amlen, see Configuring the licensed usage for Eclipse Amlen and accepting the license by using REST Administration APIs.

  1. Go to the Server Control page by selecting Server Control from the Server menu.
  2. From the IBM IoT Message Gateway Server pane, view the information that is displayed in the Licensed Usage field. This value shows the current licensed usage for Eclipse Amlen.
