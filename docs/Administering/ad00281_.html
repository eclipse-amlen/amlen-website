Restarting, stopping, and viewing the status of the Eclipse Amlen server by using the Amlen WebUI

System administrators can view the status of the Eclipse Amlen server and restart and stop the server by using the Server Control page in the Amlen WebUI.

You can view the status of the Eclipse Amlen server and restart and stop the server by using the Amlen WebUI. You can view the status of the Eclipse Amlen server and start, restart and stop the server by using REST Administration APIs. For more information about using REST Administration APIs to view the status of the server, see Viewing the status of an Eclipse Amlen server and services by using REST Administration APIs. For more information about using REST Administration APIs to start, restart, and stop the server, see Starting, restarting and stopping the Eclipse Amlen server by using REST Administration APIs.

  1. Go to the Server Control page in one of the following ways:
    • From the Home tab, click Server Control in the Customize system settings pane.
    • From the Server menu, click Server Control.
  2. Check the status of the server in the Status field of the IBM IoT Message Gateway Server pane:
    • If the server status is Running, the server is running.
    • If the server status is Maintenance, the server is in maintenance mode. For more information about maintenance mode, see Maintenance mode.
  3. Choose an action from the IBM IoT Message Gateway Server pane:
    • To restart the server, click Restart the server.

      This option is only available if the server is not running.

      If the server does not restart, check the logs for messages. For more information about the log files, see Log files.

    • To stop the server, click Stop the server. Stopping the server prevents all messaging.

      This option is only available if the server is started.
