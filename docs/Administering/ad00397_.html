Setting and viewing the admin superuser ID by using REST Administration APIs

System administrators can set and view the admin superuser ID by using REST Administration APIs.

The admin superuser ID is stored locally to ensure that the Eclipse Amlen server can be administered even if the configured external LDAP server cannot be reached, or if the external LDAP server configuration is incorrect.
