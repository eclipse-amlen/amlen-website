Configuring the MQ Connectivity SSL key repository

To send messages securely between Eclipse Amlen and IBMÂ® MQ, you must configure an SSL key repository on Eclipse Amlen.

To configure the SSL key repository you must upload a key database file and a password stash file to Eclipse Amlen. Before you can upload the files, you must first create them in IBM MQ, and add certificates to them. For more information about creating a key repository, see Configuring the IBM MQ server connection channel.

You can configure the SSL key repository by using the Amlen WebUI or by using REST Administration APIs.

For more information about configuring the SSL key repository by using the Amlen WebUI, see Configuring the MQ Connectivity key repository by using the Amlen WebUI.

For more information about configuring the SSL key repository by using REST Administration APIs, see Configuring the MQ Connectivity key repository by using REST Administration APIs.
