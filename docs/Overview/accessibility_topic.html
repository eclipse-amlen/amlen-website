Accessibility features for Eclipse Amlen

Accessibility features assist users who have a disability, such as restricted mobility or limited vision, to use information technology content successfully.

Accessibility features

The Eclipse Amlen online product documentation in IBMÂ® Knowledge Center is enabled for accessibility. The accessibility features of IBM Knowledge Center are described at http://www.ibm.com/support/knowledgecenter/about/releasenotes.html.

Accessing functions by using REST APIs

The functions that are available in the Amlen WebUI, are accessible by using REST Administration APIs. Before you can use Eclipse Amlen, you must accept the license agreement by using a REST Administration API. Until the license is accepted, the Eclipse Amlen server remains in maintenance mode. For more information about using a REST Administration API to set the licensed usage and accept the license agreement, see Configuring the licensed usage for Eclipse Amlen and accepting the license by using REST Administration APIs.

For more information about using REST APIs and the Amlen WebUI to complete administrative tasks, see Administering.

Moving around the screen elements by using the keyboard

To move around the elements on the Amlen WebUI, press Tab.

When a menu tab has focus, a box is displayed around the text for that menu tab. You can then move between the menus Home, Messaging, Monitoring, Server, Cluster and Web UI, by using the arrow keys. To go to the options that are available from the menu that is in focus, press Shift+F10. Use the space bar to select an item from the list.

To access check boxes located within a grid, press Tab to move focus to the grid. Then, use the arrow keys to move to the check box you want to select. Press F2 to focus on the check box, and press Spacebar to select or clear the check box. To move focus away from the check box, press Tab.
