MQ Connectivity

Eclipse Amlen and IBMÂ® MQ can be interconnected for use in the same solution.

The MQ Connectivity feature of Eclipse Amlen enables messages from Eclipse Amlen to be forwarded to IBM MQ, or for messages from IBM MQ to be forwarded to Eclipse Amlen.

To use MQ Connectivity, you must configure two components:
  • Queue manager connections. Each queue manager connection provides the connection details for Eclipse Amlen to connect to a IBM MQ queue manager.
  • Destination mapping rules. Each destination mapping rule provides the mapping logic between an Eclipse Amlen queue, topic, or topic subtree and a IBM MQ queue, topic, or topic subtree.

You can secure the Eclipse Amlen connection with IBM MQ by using SSL.

For more information about MQ Connectivity, see MQ Connectivity.
