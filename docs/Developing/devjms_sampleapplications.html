Sample applications

There are four sample applications for JMS: JMSSampleAdmin, JMSSample, HATopicPublisher, and HADurableSubscriber. These sample applications are in the com.ibm.ima.samples.jms package.

You can find the sample applications in the ImaClient/jms/samples path in the ImaJmsClientV1.1.zip file. This file is available on Fix Central.

Information about compiling and running the samples can be found in the IMAClient/README.txt file in the ImaJmsClientV1.1.zip file.

  • The JMSSampleAdmin application
    The JMSSampleAdmin application provides a sample tool for creating Eclipse Amlen administered objects for JMS. It demonstrates how to use ImaJmsFactory to create objects, and how to use ImaProperties to configure objects.
  • The JMSSample application
    The JMSSample application demonstrates how to send messages to and receive messages from Eclipse Amlen topics and queues.
  • The HATopicPublisher and HADurableSubscriber applications
    The HATopicPublisher and HADurableSubscriber applications demonstrate how to enable JMS client applications to use Eclipse Amlen high availability features.
