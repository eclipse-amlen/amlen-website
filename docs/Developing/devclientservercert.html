Server certificates

Server certificates must be manually installed on each client on which an SSL connection to the Eclipse Amlen server is established.

For information about securing a JMS client, see Securing Eclipse Amlen JMS client by using Java runtime environment global properties.

For information about securing MQTT clients, see the set of securing articles on the Mobile Messaging & M2M articles page of the IBM® messaging community blog.

Complete the following steps to secure a JavaScript™ client:
  1. Open a web browser.
  2. Type the following information in the URL bar:https://ipaddress:port_number and press Enter.

    Where:

    ipaddress
    The IP address where the endpoint is configured where the clients are going to connect.
    port_number
    The endpoint port.

    A warning is displayed saying that the certificate is not trusted.

  3. Follow the prompts on the warning to accept the certificate.
