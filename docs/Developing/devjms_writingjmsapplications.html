Writing JMS Applications

JMS applications are written by using the interfaces in the javax.jms package. If you already have JMS applications that you would like to reuse with Eclipse Amlen, you can do so after you have created Eclipse Amlen JMS administered objects. Point-to-point (queue) messaging is not supported for the Beta release.
