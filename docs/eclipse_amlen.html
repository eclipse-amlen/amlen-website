Eclipse Amlen

Eclipse Amlen is an Eclipse Foundation open source project. The webpage for the project can be found at https://www.eclipse.org/amlen/.

  • Product overview
    Eclipse AmlenTM delivers scalable connectivity for the Internet of Things, providing a solution to help you to connect all your devices, sensors and systems to the enterprise.
  • Planning
    When you plan your Eclipse Amlen solution, you must make a number of decisions to ensure your solution works effectively.
  • Installing Eclipse Amlen
    You can install Eclipse Amlen in Docker images or as RPMs directly on your host operating system (OS).
  • Upgrading
    System administrators can upgrade Eclipse Amlen from earlier versions or ifixes. System administrators can also upgrade to Eclipse Amlen from IBM® Watson IoT Platform Message Gateway or IBM® IoT MessageSight
  • End-to-end scenarios
    There are a number of messaging patterns that identify common message flows that are used in messaging solutions. Use the messaging patterns to help identify your messaging needs, and then use the scenarios to get up and running quickly.
  • Administering
    Administration tasks can be completed by system administrators or messaging administrators by using the Amlen WebUI, or by using REST Administration APIs.
  • Administering the Web UI
    System administrators can administer the Web UI by using the Amlen WebUI.
  • Security
    You must consider security requirements before, and after you configure Eclipse Amlen.
  • Monitoring and reporting
    Monitoring provides a picture of the health of Eclipse Amlen. You can monitor Eclipse Amlen to understand how it is being used, and watch for potential problems.
  • Backup and restore
    System administrators can create a secure backup that can be used to restore the Eclipse Amlen configuration and, optionally, store data. The backup data can be used to copy data from one server to another, or used to restore data to the same server.
  • Troubleshooting
    Look for a troubleshooting task to help you solve a problem when using Eclipse Amlen.
  • Developing client applications
    You can develop your own client applications to use with Eclipse Amlen.
  • Reference
    Use the reference information in this section to accomplish tasks associated with the use of Eclipse Amlen.
