Troubleshooting

Look for a troubleshooting task to help you solve a problem when using Eclipse Amlen.

  • Log files
    There are various system logs that you can use to help with problem determination and troubleshooting. You can set the logging level, change the log location, and view log location information by using REST Administration APIs.
  • Maintenance mode
    Maintenance mode is an Eclipse Amlen server state that allows a system administrator to complete a number of tasks. In maintenance mode, a system administrator can clean the server store, stop and start the server, and change the server state from Maintenance to Running.
  • Common problems
    There are a number of common problems that you might encounter. Your application might fail to connect. Your application might disconnect unexpectedly. Your messages might not be published, or your messages might not arrive. MQ Connectivity might not work.
  • Platform must-gather
    Eclipse Amlen version 5.0 includes a script that collects must-gather data, in both Docker and non-Docker environments. The same script is used for the messaging server and for the Web UI. System administrators can use the script to collect the must-gather diagnostic data that is required for reporting a problem to IBMÂ® support.
  • Message catalog
    You can use the messages in this catalog to help you solve problems with Eclipse Amlen.
