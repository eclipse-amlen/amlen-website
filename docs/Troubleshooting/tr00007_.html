Changing the server state

System administrators can change the server state from Maintenance to Running, or from Running to Maintenance. The server state can be changed by using either REST Administration APIs or the Amlen WebUI.

  • Changing the server state by using the Amlen WebUI
    System administrators can change the server state from Maintenance to Running, or from Running to Maintenance, by using the Server Control page in the Amlen WebUI.
  • Changing the server state by using REST Administration APIs
    System administrators can change the server state from Maintenance to Running, or from Running to Maintenance, by using REST Administration APIs.
