SNMP problems

If there is a problem with processing Simple Network Management Protocol (SNMP) queries, you might want to consider requesting less Management Information Base (MIB) data at one time.

SNMP queries fail to complete when the Eclipse Amlen server is under high load

When the Eclipse Amlen server is operating at high load or is running low on resource, SNMP queries might not be processed and you might see the following error when you request a large amount of the MIB:
Error in packet.
Reason: (genError) A general failure occurred

If you experience this problem, try requesting less MIB data at one time.
