Setting up a trace

You can set up a trace by using the Amlen WebUI.

If IBMÂ® Support requests you to set up a trace, they give you information that you must enter on the Web UI Settings page of the Amlen WebUI.

  1. Click Web UI Settings in the Appliance menu.
  2. Expand the Service section.
  3. Select the Enable trace check box.
  4. Type the string provided by IBM Support in the Trace Specification field.
  5. Click Save.
IBM Support will run the trace to determine the problem.
After IBM Support has the information they need, return to the Web UI Settings page and clear the Enable trace check box. Then click Save to disable the trace.
