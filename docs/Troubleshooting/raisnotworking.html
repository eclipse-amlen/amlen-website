Eclipse Amlen resource adapter is not working

A number of tools are provided within Eclipse Amlen and within WebSphereÂ® Application Server that you can use to help with problem determination and troubleshooting.

When troubleshooting Eclipse Amlen resource adapter problems, a number of tools are provided in Eclipse Amlen that you can use to diagnose the problem:

  • Log files. System logs are generated to help with problem determination and troubleshooting. For more information about log files, see Log files.
  • Common problems. There are a number of common problems that you might encounter. For more information about common problems, see Common problems.
  • Message catalog. Use the messages in this catalog to help you solve problems with Eclipse Amlen. For more information about message catalog, see Message catalog.

For more information about troubleshooting in WebSphere Application Server including problem determination tools that are provided with the product, see Troubleshooting and support in the WebSphere Application Server documentation.
