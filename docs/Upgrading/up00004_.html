Rolling back the level of Eclipse Amlen in a Docker environment

If, after an upgrade of the software level of Eclipse Amlen in a Docker environment, a rollback to a prior level is required, system administrators can perform the rollback operation.

Follow the procedure that is described in Upgrading Eclipse Amlen in a Docker environment but ensure that you specify a fix pack that is earlier than the currently installed Eclipse Amlen version.
