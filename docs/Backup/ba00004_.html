Backup and restore in a high availability environment

The backup and restore of a high availability (HA) environment needs to be a coordinated process between the servers in an HA pair.

The process of backing up and restoring an HA environment requires the use of a single backup file. For information about creating a secure backup of an HA environment, see Creating a backup of the Eclipse Amlen server in a high availability environment. For information about restoring the HA environment, see Restoring the high availability environment.
