Show TopicMonitor

URI

Shows all topic monitors on Eclipse Amlen.

URI

Use the Eclipse Amlen REST API GET method with the following Eclipse Amlen configuration URI:

http://<admin-endpoint-IP:Port>/ima/v1/configuration/TopicMonitor

Parameters

None.

Usage NotesÂ®

  • The command must be capitalized as shown.

Related Commands

  • Delete TopicMonitor
  • Show topic related statistics by using REST Monitoring APIs

Example

Lists all topic monitors:
curl -X GET http://127.0.0.1:9089/ima/v1/configuration/TopicMonitor
Example response to GET method:

{
  "Version": "v1",
  "TopicMonitor": []
}
