Message hub REST Administration APIs

The message hub REST APIs allow you to configure message hubs, endpoints, connection policies, and messaging policies. You can create, update, view, and delete message hubs, endpoints, connection policies, and messaging policies.

For more information about viewing message hubs, endpoints, connection policies, and messaging policies see Show configuration details of nameable configuration objects.

For more information about deleting message hubs, endpoints, connection policies, and messaging policies, see Delete configuration objects .
