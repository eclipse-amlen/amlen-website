REST API Reference

The REST API REST Administrative API reference section provides reference topics for the REST APIs that are available for Eclipse Amlen.

Running a REST API

Eclipse Amlen has a REST (REpresentational State Transfer) interface and provides REST APIs which you can use to administer and monitor your system.

To run a REST API, call a method on a REST resource and pass parameters or a request in JSON format. The methods that the REST APIs use are POST, GET, and DELETE:
  • The POST method creates or updates a configuration object.
  • The GET method retrieves data about a configuration object.
  • The DELETE method deletes a configuration object.

You can write scripts or use the Linuxâ„¢ program cURL to run the REST APIs. Combine a REST URI with the REST method to be used and the data to be passed to create the REST request.

The REST APIs

The following sets of REST APIs are supported in Eclipse Amlen:

  • Delete configuration objects
  • Show configuration details of nameable configuration objects
  • Show configuration details of fixed-name configuration objects
  • Copy a file from the local machine to Eclipse Amlen
  • Admin endpoint REST Administration APIs
    The admin endpoint REST Administration APIs allow you to configure and display admin endpoint,configuration policy, and admin superuser ID settings. You can also set the admin superuser password.
  • Server REST Administration APIs
    The server REST APIs are used to stop, start, restart, and view the status of the Eclipse Amlen server. You can also set the logging level.
  • Cluster REST Administration APIs
    The cluster REST Administration APIs allow you to configure and display cluster membership settings.
  • Security REST Administration APIs
    The security REST Administration APIs allow you to configure security on Eclipse Amlen. You can import keys and certificates to secure connections to the system, and enable or disable FIPS-compliant security.
  • Message hub REST Administration APIs
    The message hub REST APIs allow you to configure message hubs, endpoints, connection policies, and messaging policies. You can create, update, view, and delete message hubs, endpoints, connection policies, and messaging policies.
  • Queue REST Administration APIs
    The queue REST Administration APIs allow you to configure message queues. You can create, edit, view and delete queues.
  • MQ Connectivity REST Administration APIs
    The MQ Connectivity REST Administration APIs allow you to configure MQ Connectivity. You can create, edit, view, and delete destination mapping rules and queue manager connections. You can enable and disable MQ Connectivity. You can start, restart and stop the MQ Connectivity process.
  • High availability REST Administration APIs
    The high availability (HA) REST Administration APIs allow you to configure HA. You can view and edit the current HA settings.
  • Backup and restore commands
    Commands that you can use to export and import Eclipse Amlen server configuration data, and also to reset the server to a factory-new state.
  • REST Monitoring APIs
    REST Monitoring APIs allow you to monitor Eclipse Amlen. You can monitor topics, subscriptions, connections, endpoints, destination mapping rules, MQTT clients, queues, transactions, memory usage, and store usage.
  • XA transaction REST Administration APIs
    The XA transaction REST Administration APIs allow you to heuristically commit, roll back, or forget an XA transaction branch.
  • SNMP REST Administration APIs
    The SNMP REST Administration APIs allow you to configure the Simple Network Management Protocol (SNMP) on Eclipse Amlen. You can enable and disable SNMP monitoring. You can display the status of SNMP. You can start, stop and restart SNMP.
  • Protocol plug-in REST Administration APIs
    The protocol plug-in REST Administration APIs allow you to configure and show protocol plug-ins, the protocol plug-in properties file, the protocol plug-in server host IP address or host name, the protocol plug-in server port number, the protocol plug-in debug server host IP address or host name, the protocol plug-in debug port number, the maximum heap size of the JVM remote debugger, and runtime parameters of the protocol plug-in server JVM. You can view protocol plug-ins and protocols that are on the Eclipse Amlen server. You can also start, stop, and restart the protocol plug-in server.
  • Special purpose REST Administration APIs
    The special purpose REST Administration APIs take a specific action on the server or display specific information.
