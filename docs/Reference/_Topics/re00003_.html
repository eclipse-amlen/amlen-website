Supported cipher lists

You can use ciphers to specify the encryption algorithm that is used by the security policy.

There are different qualities of cipher available. You can choose from the following quality of ciphers:
  • Best: The most secure cipher that is supported by the server and the client.
  • Fast: The fastest high security cipher that is supported by the server and the client.
  • Medium: The fastest medium or high security cipher that is supported by server and the client.

For more information about ciphers, see Transport Layer Security
