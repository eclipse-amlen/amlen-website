Viewing endpoint level statistics by using the Amlen WebUI

Use the endpoint monitor to view live, aggregated endpoint data, and to query the best and worst performing connections across several connection metrics.

You can view endpoint monitoring statistics by using the Amlen WebUI, by using a REST Monitoring API, or by using an external monitoring application. For more information about using a REST Monitoring API to view endpoint monitor statistics, see Viewing endpoint level statistics by using REST Monitoring APIs. For more information about using an external application to view endpoint monitor statistics, see Viewing endpoint level statistics by using an external application. For more information about the predefined queries that you can run, see Endpoint monitoring.

From the Amlen WebUI, navigate to the Endpoints page by selecting Monitoring > Endpoints.
  • To view time-series charts for a particular endpoint, metric, and time duration, complete the following steps:
    1. From the Endpoints page, go to the Endpoint Chart pane.
    2. In Source, select the name of the endpoint that you want to monitor.
    3. In Show, select a predefined query that you want to filter on.
    4. In Duration, select the period that you want to monitor.
    5. Click Refresh to generate the output in the endpoint monitor table.
  • To view statistics for a particular endpoint and metric, complete the following steps:
    1. Go to the Endpoint Data pane. Endpoints are displayed in the Endpoint data table. Up to 10 records can be displayed in the table.
    2. In Query, select the predefined query that you want to filter on.
    3. Optional. Enter a value into Filter to change the visibility of the data that is shown in the table. The filter is applied to the maximum of items that are returned by the query.
    4. Click Refresh to generate the output in the endpoint monitor table.

The endpoint monitor table is updated to display the results of the query and any filter that is applied to those results.

You can view the total number of items that are returned in the Total field. You can see the number of items that are selected in the Selected field. You can move through the table of results by using Next Page and Previous Page or Go to a specific page. You can change the number of items to view per page by selecting from the available options.
