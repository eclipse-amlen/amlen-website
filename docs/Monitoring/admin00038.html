Viewing server statistics by using the Amlen WebUI

Use the Server Dashboard to view statistics about Eclipse Amlen server.

You can view Eclipse Amlen server statistics by using the Amlen WebUI, or by using a REST Monitoring API. For more information about using a REST Monitoring API to view Eclipse Amlen server statistics, see Viewing server statistics by using a REST Monitoring API.

From the Amlen WebUI, you can view information about the Eclipse Amlen server by using the Server Dashboard.
  • To view the Server Dashboard select Home from the Amlen WebUI main menu. Use the Quick Stats information to get an overview of server connections and performance.
