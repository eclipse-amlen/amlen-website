Viewing queue related statistics by using the Amlen WebUI

Use the queue monitor to view live, aggregated queue data, and to query the best and worst performing queues across several connection metrics.

You can view queue monitoring statistics by using the Amlen WebUI, or by using REST Administration APIs. For more information about using REST Administration APIs to view queue monitoring statistics, see Viewing queue-related statistics by using REST Monitoring APIs. For more information about the predefined queries that you can run, see Queue Monitoring.

  1. From the Amlen WebUI, navigate to the Queues page by selecting Monitoring > Queues.
  2. In Queue Name, enter the value of the queue you want to monitor. You can enter one of the following values:
    • A specific queue name.
    • The asterisk sign (*) to select all queues.
    • Part of a queue name in combination with the asterisk sign (*) to select a subset of queues.
  3. In Query, select the predefined query that you want to filter on.
  4. Click Refresh to generate the output in the queue monitor table.
  5. Enter a value into Filter to change the visibility of the data that is shown in the table. The filter is applied to the maximum of items that are returned by the query.

The Timestamp is updated to show the time that query is generated.

The table is updated with the results of the query and any filter that is applied to those results.

You can view the total number of items that are returned in the Total field. You can see the number of items that are selected in the Selected field. You can move through the table of results by using Next Page and Previous Page or Go to a specific page. You can change the number of items to view per page by selecting from the available options.
