Viewing memory statistics by using the Amlen WebUI

Use the Server monitor to view statistics about Eclipse Amlen memory.

You can view statistics about Eclipse Amlen memory by using the Amlen WebUI, by using a REST Monitoring API, or by using an external monitoring application. For more information about using a REST Monitoring API to view Eclipse Amlen memory statistics, see Viewing memory statistics by using REST Monitoring APIs. For more information about using an external application to view Eclipse Amlen memory statistics, see Viewing memory by using an external application. For more information about the statistical data that you can view, see Memory Monitoring.

  1. From the Amlen WebUI, select Monitoring > Server to view memory statistics.
  2. Click Refresh to update the statistical information displayed.

The statistical data in the table is updated.

The Timestamp is updated to show the time that refreshed data is generated.
