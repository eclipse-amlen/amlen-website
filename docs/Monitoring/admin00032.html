Viewing MQ Connectivity statistics by using the Amlen WebUI

System administrators can view destination mapping rule-related statistics by using the MQ Connectivity monitor in the Amlen WebUI.

You can view destination mapping rule statistics by using the Amlen WebUI, or by using a REST Monitoring API. For more information about using a REST Monitoring API to view destination mapping rule statistics, see Viewing MQ Connectivity statistics by using REST Monitoring APIs. For more information about the predefined queries that you can run, see MQ Connectivity monitoring.

You can use the hover help over Status to view more information about why a particular rule might be unavailable or restarting. A status of Enabled usually indicates that the rule is operating normally. However, it is possible that messages might not be published for other reasons. For example, a connected client might not be authorized to publish on a particular topic.

  1. From the Amlen WebUI, select Monitoring > MQ Connectivity.
  2. Enter a value into Rule Name. The default value is *.
  3. Enter a value into Queue Manager Connection. The default value is *.
  4. Enter a value into Rule Type. The default value is Any.
  5. Enter a value into Query to select the predefined query that you want to run. The default value is Rules with most commit operations.
  6. Click Refresh to generate the output in the table.
  7. You can change the number of rules that are added to the table by selecting a value of 10, 25, 50, or 100.
  8. Enter a value into Filter to change the visibility of the data that is shown in the table. The filter is applied to the maximum of items that are returned by the query.

The Timestamp is updated to show the time that query is generated.

The table is updated to display the results of the query and any filter that is applied to those results.

You can view the total number of items that are returned in the Total field. You can see the number of items that are selected in the Selected field. You can move through the table of results by using Next Page and Previous Page or Go to a specific page. You can change the number of items to view per page by selecting from the available options.
