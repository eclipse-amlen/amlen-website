Viewing store statistics by using the Amlen WebUI

Use the Server Dashboard or the Server Monitor to view statistics about Eclipse Amlen persistent memory.

You can view Eclipse Amlen store statistics by using the Amlen WebUI, by using a REST Monitoring API, or by using an external monitoring application. For more information about using a REST Monitoring API to view Eclipse Amlen store statistics, see Viewing store statistics by using a REST Monitoring API. For more information about using an external application to view Eclipse Amlen store statistics, see Viewing store statistics by using an external application. For more information about the statistical data that you can view, see Store monitoring.

From the Amlen WebUI, you can view information about Eclipse Amlen persistent memory by using the Server Dashboard and the Server Monitor.
  • To view the Server Dashboard select Home from the Amlen WebUI main menu. Use the Server Memory Usage graph to understand persistent memory usage over a period of time.
  • To view the Server Monitor, select Monitoring > Server from the Amlen WebUI main menu. Use the information to understand a current snapshot of persistent store statistics. Click Refresh to update the statistical information displayed. The Timestamp is updated to show the time that refreshed data is generated.
