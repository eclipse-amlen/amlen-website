File paths

You might need to access certain files to locate specific information. For example, the log and trace files are required to keep a record of any issues when the Bridge is running.

The log file contains external information and records errors, warnings, and notices.

The trace file contains internal information that might be needed to debug any problems that are found.

The following is a list of important files and the directories that they are stored in:

  • Bridge installation directory: /opt/ibm/imabridge.
  • Executable file and scripts: /opt/ibm/imabridge/bin
  • Libraries: /opt/ibm/imabridge/lib64
  • Http-related files: /opt/ibm/imabridge/http

The variable data is stored in the following directories:

  • Default working directory: /var/imabridge
  • Dynamic config file: /var/imabridge/bridge.cfg
  • Keystore: /var/imabridge/keystore
  • Truststore: /var/imabridge/truststore
  • Sample config files: /var/imabridge/samples
  • Log file default location: /var/imabridge/diag/logs/imabridge.log
  • Trace file default location: /var/imabridge/diag/logs/imabridge.trace
