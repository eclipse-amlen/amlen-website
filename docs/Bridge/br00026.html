Installing Amlen Bridge

You can install Amlen Bridge in Docker containers or as RPMs directly on your host operating system. On Red Hat systems, you must have an active Red Hat subscription to proceed. You do not need such a subscription on CentOS.

After you install the Amlen Bridge, you must set the license type and accept the license agreement by using a REST Administration API. Until the license is accepted, the Amlen Bridge will not connect to MQTT endpoints.

For more information about using a REST Administration API to set the licensed usage and accept the license agreement, see Accepting the Amlen Bridge license.
